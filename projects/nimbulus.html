Nimbulus Webclock Mobile App

This is the login screen for a mobile app I designed for a client who wanted a timecard mobile application. Users could clock in and clock out using their phones with this app. The app is written in Nativescript, which allows the framework to be written once and deployed for both iOS and Android. I designed all of the layout, and I also developed several features of this application including a working sidebar, giving password prompts, and creating an interactive user interface.
