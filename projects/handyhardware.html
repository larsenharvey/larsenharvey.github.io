Rebranding - Handy Hardware

This is a rebranding project for a local hardware store in Rock Island, Illinois. The goals for this project were to create a new and improved logo (the previous logo can be found here, and to have the hardware store appeal to a younger generation. To do this, I mocked up an app concept for this hardware store, using Adobe XD. The full interactive prototype can be found here.
